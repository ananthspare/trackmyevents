Category Todos Functionality Test

This is a simple test to verify the category todos functionality is working correctly.

Steps to test:

  1. Run the application: npm start
  2. Navigate to the Categories page
  3. Select a category
  4. Look for the "Category To-dos" section in the right panel before the events list
  5. Type a todo in the input field and press Enter or click the + button
  6. The todo should appear in the list below
  7. Click the checkbox to mark todos as complete/incomplete
  8. Click the trash icon to delete todos
  9. Refresh the page and select the same category - todos should persist

Expected behavior:

  • Category To-dos section appears between category header and events list
  • Todos can be added, completed, and deleted
  • Todos persist after page refresh
  • Todos are stored in DynamoDB with categoryID field
  • Each category/subcategory has its own separate todo list
